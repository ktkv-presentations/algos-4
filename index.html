Типы данных

Все типы данных можно разделить на

  • изменяемые (множества, списки, словари);
  • неизменяемые (кортежи, строки, числа);
  • упорядоченные (списки, строки, кортежи, словари);
  • неупорядоченные (множества).

Комментарии

Числовые типы

int

Целые числа
Поддерживается произвольная длина чисел

float

Числа с плавающей точкой
Представляют вещественные числа (дробные)

complex

Комплексные числа
представляются в виде a + bj, где a — действительная часть, а b — мнимая

Логический тип

bool

может принимать одно из двух значений True и False

Строковый тип

str

используются для представления текста и представляют собой последовательность символов

Список

list

упорядоченная изменяемая коллекция объектов

Список

list

упорядоченная изменяемая коллекция объектов

Кортеж

tuple

Как списки, но неизменяемы (immutable)
Создают фиксированную последовательность данных

Множество

set

неупорядоченная коллекция уникальных элементов (нет дублирующихся значений)

Словарь

dict

неупорядоченная коллекция пар «ключ-значение», где ключи - уникальны

None

обозначает отсутствие значения или пустое значение

bytes

неизменяемая последовательность байтов и используется для работы с бинарными данными

Операции и выражения в Python

Операнд - элемент, над которым выполняется операция в выражении

Операции - действие, которое выполняется над значениями или объектами

Выражение - комбинация операндов и операторов, которая при вычислении возвращает результат

Арифметические операции

используются для выполнения математических вычислений

result = 5 + 3 # 8 - сложение

result = 10 - 4 # 6 - вычитание

result = 6 * 7 # 42 - умножение

result = 8 / 2 # 4.0 - деление

result = 9 // 2 # 4 - деление нацело

result = 10 % 3 # 1 - остаток от деления

result = 2 ** 3 # 8 - возведение в степень

Арифметика со строками

Операции сравнения

5 = = 5 # True - равно

5 ! = 3 # True - не равно

7 > 3 # True - больше

3 < 7 # True - меньше

7 > = 7 # True - больше или равно

3 < = 7 # True - меньше или равно

==, >= и так далее пишутся вместе

Присваивание и составные операторы

x = 10 # 10 - присваивание

x += 10 # 20 - добавляет и присваивает

x -= 5 # 15 - вычитает и присваивает

Логические операции

True and True # True - логическое и

False or True # True - логическое или

not False # True - логическое не

Битовые операции

5 & 3 # 1 - побитовое и

5 | 3 # 7 - побитовое или

~5 # -6 - побитовое не

Приоритет операций

Высокий приоритет

** * / // % % + -

Низкий приоритет

== != > < >= <= and or not

Скобки и порядок вычисления

для изменения приоритета операций можно использовать круглые скобки

Левостороннее и правостороннее связывание

Некоторые операторы имеют левостороннее связывание, то есть они выполняются слева направо, а другие — правостороннее связывание

Структура программы

программа состоит из набора инструкций, которые выполняются последовательно

Базовая структура программы

Состоит из одной или нескольких функций

Функция

позволяют разбить программу на логически связанные блоки кода, которые могут быть вызваны после её объявления

В python нет hoisting

Касается переменных тоже

уникальная особенность ЯП, которая позволяет вам обращаться к переменным и функциям до того, как они были объявлены в коде
Зато есть в Golang, Ruby, JavaScript, VBScript

Шаблонизация строк

позволяют динамически вставлять данные в строки

Модули и импорт

Модули позволяют организовать код в отдельные файлы для улучшения его структурированности
Импорт обычно производится в самом начале программы

Управляющие конструкции

Программы могут включать в себя условные операторы, циклы и обработку исключений, чтобы контролировать выполнение различных блоков кода в зависимости от условий

Особенности условных конструкций

Ветвление с помощью if, elif, else
Неограниченное чисто elif
Блок else необязателен

Условная конструкция match

используется для сравнения значения с несколькими шаблонами

Циклы

конструкции, которые позволяют многократно выполнять набор инструкций, пока соблюдается определённое условие

Цикл for

используется для прохода по элементам итерируемого объекта (списка, строки, кортежа, множества или словаря)

Цикл while

выполняет инструкции до тех пор, пока заданное условие остаётся истинным (True)

Операторы управления циклами

специальные операторы, которые позволяют управлять выполнением цикла

Оператор break

используется для немедленного завершения работы цикла, даже если его условие ещё истинно

Оператор continue

пропускает текущую итерацию цикла и переходит к следующей, не завершая цикл полностью

Оператор else

блок выполняется, когда цикл завершился естественным образом, то есть не был прерван оператором break

Цикл for с функцией range()

range(n) генерирует последовательность чисел, которую можно использовать для итераций
range(n)
последовательность чисел от одного до n-1
range(b, n)
последовательность чисел от b до n-1
range(b, n, s)
последовательность чисел от b до n-1 с шагом s

Бесконечные циклы

если условие в цикле while всегда остаётся истинным, цикл становится бесконечным

Вложенные циклы

один цикл может находиться внутри другого
